Keyboard Shortcuts

The following keyboard shortcuts are currently implemented:

  • CTRL + z undo
  • CTRL + y | CTRL + SHIFT + Z redo
  • ESC open main menu
  • ↑ | + zoom-in
  • ↓ | - zoom-out
  • CTRL + r reset zoom
  • ↹ toggle active figure when drawn / toggle initiative input when focused input
  • n next round / draw
  • m draw AM card (deck depending on active figure, monster deck when not active, disabled when Draw)
  • l draw from Loot Deck
  • 1 - : toggle through element state
  • h toggle Hide absent figures
  • s select figure/standee for opening menu with number key (0-9)
  • d difficulty/level menu
  • e event effects menu
  • f (finish) scenario menu
  • p open party sheet

On any open figure/standee menu, those shortcuts are disabled, currently shortcuts available in figure/standee menu:

  • ← dec. current HP
  • → inc. current HP
  • 0-9 toggle condition/condition modifier
  • b inc. Bless
  • SHIFT + B dec. Bless
  • c inc. Curse
  • SHIFT + C dec. Curse
  • k | d kill/exhaust
  • ↑ inc. max. HP
  • ↓ dec. max. HP
  • s open character sheet

On open menu:

  • ↵ | SPACE confirm action / open submenu
  • ESC decline action / close menu
  • ↹ select action
