# AI Notice

## AI in This Project

AI tools (such as GitHub Copilot, ChatGPT, Claude, or similar) are used during the development of _Gloomhaven Secretariat_ as productivity aids. This includes tasks such as generating boilerplate code, suggesting implementations, drafting documentation, and reviewing logic.

All AI-assisted output is reviewed, verified, and ultimately the responsibility of the author. AI is used as a tool — not as a replacement for understanding the codebase or the game rules. Game data in particular is always verified against the physical game components before being committed.

## AI in Contributions

AI usage is also permitted for contributors. Transparency is required: any AI-assisted contribution must be disclosed in the pull request description.

See the [AI Usage Policy](./CONTRIBUTING.md#ai-usage-policy) in the Contributing Guide for full details.

Do not submit AI-generated code if the terms of your AI tool are incompatible with AGPL-3.0 or require you to retain rights that conflict with open-source distribution.

AI must not be used to reproduce, reverse-engineer, or infer copyrighted game content beyond what is explicitly covered by the licenses already acknowledged in this repository (see [Copyright / License](./README.md#copyright--license) in the README).
